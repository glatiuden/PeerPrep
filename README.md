# PeerPrep

This repository contains all the development files of PeerPrep - an interview preparation platform where students can find peers to practice whiteboard-style technical questions.

## Tech Stacks

- Nuxt.js (Vue.js)
- Node.js + Express.js
- MongoDB (Atlas)
- Docker
- Kubernetes
- AWS (Production)
  - ECR
  - EC2
  - EKS
  - ALB
  - S3
  - Cloudfront
  - Route53
  - Elasticache (Redis)
- GCP
  - GKE
- CloudAMQP (RabbitMQ)
- GitHub Actions (CI)
- Flux V2 (CD)
- Prometheus
- Grafana
- Loki + Promtail
- Terraform
